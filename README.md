# NEXUS CITY - Underground Portal

This repository contains a single-page, interactive, cyberpunk-themed webpage. It is built purely with HTML, CSS, and vanilla JavaScript, showcasing a rich, animated user interface inspired by cyberpunk aesthetics.

## üöÄ Live Demo

You can view the live page by simply opening the `index.html` file in your browser.

## ‚ú® Features

*   **Immersive Cyberpunk UI**: Dark, neon-heavy design with animated backgrounds, scanlines, and particle effects.
*   **Interactive Sections**: The page is divided into several thematic sections accessible via a smooth-scrolling top menu:
    *   Districts
    *   Classified Locations
    *   Street Language (Slang)
    *   Culinary Black Market (Food)
    *   System Analytics Dashboard
    *   Survival Protocols
*   **Search Functionality**: An integrated search bar allows you to quickly find and navigate to different sections or uncover hidden content.
*   **Secret Panels**: Discover hidden "secret" panels through specific search terms (`ADMIN`, `SPIRE`, `SURVEILLANCE`) or keyboard shortcuts:
    *   `Ctrl + Shift + X`: Access the "SYSTEM OVERRIDE" panel.
    *   `Ctrl + Shift + S`: Access the "RESISTANCE NETWORK" panel.
*   **Responsive Design**: The layout adjusts for smaller screen sizes, making it accessible on different devices.
*   **Custom Cursor**: A sci-fi themed cursor enhances the interactive experience.

*   **HTML5**: For the structure and content of the page.
